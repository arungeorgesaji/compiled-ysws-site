Compiled YSWS

Spent at least 10 hours building a compiled language and we will ship you a 40$ grant for buying electronics

Language Requirements

You can design your language however you want but it should feature these essential elements(P.S: The compiler should be written in a low level language like C/C++/Rust):

Variables

Support for declaring, assigning, and using variables(maybe add scoping too).

Operators

Mathematical, logical, and comparison operators for building expressions.

Functions

Ability to define and call functions(maybe add parameters and return values too).

Error Handling

Mechanisms to detect, report, and handle errors gracefully during compilation and execution.

If you make a web playground for your compiled language you get $5 extra grant (I recommend the usage of containers to connect the language up)

I expect everyone to make a source-source language. If anyone makes a compiled language which can directly compile to assembly you might get something special
